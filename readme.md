# Chronos.P2P
实现P2P通讯的服务器与客户端

## 功能
利用基于UDP协议的P2P打洞技术，实现客户端建立连接，并可进行
- 发送消息
- 快速传送文件
## modules
### [client](./src/Chronos.P2P.Client)
客户端与服务器实现
### [sample](./src/Chronos.P2P.Server.Sample)
示例
### [test](./src/Chronos.P2P.Test)
测试
## Build From Source  

请确认下载最新版本的vs2019，并且勾选了.Net跨平台工作负载
然后打开.sln文件，即可编译、运行
